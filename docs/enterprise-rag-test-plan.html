1.3 代码 AST 切分（多语言）

验证 Python 和 JS 均能按函数/类切分并保留父级上下文

cd backend && uv run python -c "
import asyncio, tempfile, os
from app.rag.document_handler.code_processor import CodeProcessor

async def test_lang(ext, code):
    with tempfile.NamedTemporaryFile(mode='w', suffix=ext, delete=False) as f:
        f.write(code); tmp = f.name
    docs = await CodeProcessor().process(tmp); os.unlink(tmp)
    print(f'{ext}: {len(docs)} chunks')
    for d in docs:
        print(f'  [{d.metadata.get(\"node_type\",\"?\")}] {d.page_content[:60]}')
    return docs

async def main():
    await test_lang('.py', '''
class UserService:
    def login(self, u, p): return True
    def logout(self, u): pass
''')
    await test_lang('.js', '''
class UserService {
    login(u, p) { return true; }
    logout(u) { return false; }
}
function main() { console.log(\"start\"); }
''')
asyncio.run(main())
"
验证项企业级要求通过
Python 切分≥ 2 chunk（class + 方法），含 class: UserService 上下文⬜
JS 切分≥ 2 chunk，含 class: UserService⬜
函数名保留含 def: login 和 def: main⬜
Fallback 不崩溃语法错误退化为空行分块⬜

2.4 多因素排序（时间衰减 + 权重）

验证相同相关性下新文档排前面、类别权重正确生效

cd backend && uv run python -c "
import asyncio, time
from langchain_core.documents import Document
from app.rag.multi_factor_ranker import MultiFactorRanker
async def test():
    now = int(time.time())
    yr = now - 365*24*3600
    docs = [
        Document(page_content='旧文档', metadata={'created_at':yr,'doc_weight':1,'md5':'old'}),
        Document(page_content='新文档', metadata={'created_at':now,'doc_weight':1,'md5':'new'}),
    ]
    result = await MultiFactorRanker().rank('查询', docs, [0.8, 0.8])
    for i,d in enumerate(result): print(f'{i+1}. {d.page_content}')
    assert result[0].metadata['created_at'] == now
    print('时间衰减验证通过 — 新文档排在第一')
asyncio.run(test())
"
验证项企业级要求通过
时间衰减相同相关性下，新文档 > 旧文档⬜
类别权重政策类 (1.0) > 会议纪要 (0.3)⬜
质量评分ingest 后自动计算⬜
