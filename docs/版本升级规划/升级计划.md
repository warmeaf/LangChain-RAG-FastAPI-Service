# 升级计划

1. 基于聊天记录 `docs/版本升级规划/与ChatGPT的聊天记录.md` 手写一份升级计划；
2. 在 docker 容器中，再初始化一个新的基础设施；
3. 基于 grill-me skill，拷问升级计划，确保升级计划需求明确、方向无误、可执行、可验证；
4. 基于 /goal 命令，持续运行，直到完成升级。

## 升级计划原文

将当前 RAG 检索系统，进行全面升级，升级成基于 AI Agent 的内容检索系统，核心是 Agent loop，有一套周密的系统提示词，交代有哪些可用的工具，为了完成检索任务，可自行选择调用的工具，这些工具包括：
1. RAG / Vector Search 工具
2. Keyword Search 工具
3. SQL 查询工具
4. Metadata Filter 工具
5. Analytics Tool 工具
6. Code Interpreter 工具
7. API Tool 工具（检索天气、地图、航班等）
8. Time Tool 工具（获取时间）
9. OCR Tool 工具（orc 识别）
