# 升级计划

升级计划将会完全交给 AI Agent 去执行，所以必须确保以下升级计划正文，对 Agent 而言满足以下要求：
1. 需求明确；
2. 方向无误；
3. 可执行落地；
4. 可自行验证，

## 升级计划正文

将当前 RAG 检索系统，进行全面升级，升级成基于 AI Agent 的内容检索系统，核心是 Agent loop，有一套周密的系统提示词，交代有哪些可用的工具，为了完成检索任务，可自行选择调用的工具，这些工具包括：
1. RAG / Vector Search 工具
2. Keyword Search 工具
3. SQL 查询工具
4. Metadata Filter 工具
5. Analytics Tool 工具
6. Code Interpreter 工具
7. API Tool 工具（检索天气、地图、航班等）
8. Time Tool 工具（获取时间）
9. OCR Tool 工具（OCR 识别）

消息格式，工具调用规范和数据结构遵循 Anthropic 的规范；
严格基于 TDD 开发；
自行访问数据库等基础设施，自行验证数据是否正确；
不用担心破坏性重构，一切为了最终实现升级目标；
依赖可使用当前已有的，也可以安装新的。
